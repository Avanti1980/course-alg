再看最大子数组

跨越中点的最大子数组涉及求以$A[mid]$作结尾的最大子数组

设$dp[i]$为以$A[i]$作结尾的最大子数组的和，递推关系

$$ \begin{align*} \quad dp[i] = \begin{cases} dp[i-1] + A[i], & dp[i-1] \ge 0 \\ A[i], & dp[i-1] < 0 \end{cases} \end{align*} $$

  • 如果$dp[i-1] \ge 0$，把$A[i]$直接接在$dp[i-1]$表示的那个数组的后面
  • 如果$dp[i - 1] < 0$，加上前面的数反而更小了，干脆“另起炉灶”
再看最大子数组

设$dp[i]$为以$A[i]$作结尾的最大子数组的和，递推关系

$$ \begin{align*} \quad dp[i] & = \begin{cases} dp[i-1] + A[i], & dp[i-1] \ge 0 \\ A[i], & dp[i-1] < 0 \end{cases} = \max \{ dp[i-1] + A[i], A[i] \} \end{align*} $$ 

def find_max_subarray_dp(A):
    dp = [0] * len(A)  # dp[i]为以A[i]作结尾的最大子数组的和
    dp[0] = A[0]
    for i in range(1, len(A)):
        dp[i] = max(dp[i-1] + A[i], A[i])  # 状态转移
    return max(dp)

只有一重 for 循环，时间复杂度$T(n) = \Theta(n)$

动态规划

动态规划一般用来求解最优化问题：求问题的最大值、最小值

  • 最大连续子数组
  • 最长公共子序列
  • 最长递增子序列
  • 最优二叉搜索树

动态规划的一般步骤：

  • 分析最优解的结构特征
  • 递归定义最优解的值
  • 计算最优解的值
  • 利用计算出的信息，构造一个最优解 (可选)

def find_max_subarray_dp2(A):
    dp, s = [0] * len(A), [0] * len(A)  # dp[i]为以A[i]作结尾的最大子数组的和
    dp[0] = s[0] = A[0]  # s[i]为以A[i]作结尾的最大子数组的起始索引
    for i in range(1, len(A)):
        if dp[i-1] >= 0:
            dp[i] = dp[i-1] + A[i]
            s[i] = s[i-1]
        else:
            dp[i] = A[i]
            s[i] = i
    max_value = max(dp)
    max_index = dp.index(max_value)
    return s[max_index], max_index, max_value
